# 7. Climate Impact Credits (CIC)

## 7.1 Overview

Climate Impact Credits (CIC) are the native digital tokens of the AI-G-DAO ecosystem, representing verified environmental contributions and enabling democratic participation in climate governance. Built on the carbon-negative Algorand blockchain with live TestNet deployment (Contract ID: 744174033), CICs reward meaningful climate action while powering our decentralized governance system through real blockchain transactions.

### 7.3.2 Credit Calculation Algorithm

```typescript
interface CreditCalculationParams {
  // AI Analysis Results
  overallAIScore: number;           // 0-100
  environmentalImpact: number;      // 0-100
  feasibilityScore: number;         // 0-100
  riskScore: number;                // 0-100 (lower is better)
  
  // Impact Metrics
  carbonReduction: number;          // Tons CO2 equivalent
  biodiversityScore: number;        // 0-100
  pollutionReduction: number;       // Quantified reduction
  resourceEfficiency: number;       // 0-100
  
  // Project Characteristics
  projectScale: 'local' | 'regional' | 'national' | 'global';
  implementationTime: number;       // Months
  verificationLevel: 'self' | 'community' | 'third-party';
  
  // Bonus Factors
  innovationBonus: boolean;         // Novel approach
  educationComponent: boolean;      // Educational value
  communityEngagement: number;      // Community participation level
}

function calculateCICReward(params: CreditCalculationParams): number {
  // Base calculation from AI scores
  const aiComponent = (
    params.overallAIScore * 0.4 +
    params.environmentalImpact * 0.3 +
    params.feasibilityScore * 0.2 +
    (100 - params.riskScore) * 0.1
  );
  
  // Impact multipliers
  const impactMultiplier = 1 + (
    (params.carbonReduction * 0.001) +        // 0.1% per ton CO2
    (params.biodiversityScore * 0.002) +      // 0.2% per biodiversity point
    (params.pollutionReduction * 0.001) +     // Variable based on type
    (params.resourceEfficiency * 0.001)       // 0.1% per efficiency point
  );
  
  // Scale multipliers
  const scaleMultipliers = {
    local: 1.0,
    regional: 1.2,
    national: 1.5,
    global: 2.0
  };
  
  // Verification multipliers
  const verificationMultipliers = {
    self: 1.0,
    community: 1.2,
    'third-party': 1.5
  };
  
  // Time bonus (faster implementation = higher reward)
  const timeBonus = Math.max(0.8, 1.5 - (params.implementationTime / 12));
  
  // Innovation and engagement bonuses
  const innovationBonus = params.innovationBonus ? 1.1 : 1.0;
  const educationBonus = params.educationComponent ? 1.05 : 1.0;
  const engagementBonus = 1 + (params.communityEngagement * 0.001);
  
  // Final calculation
  const baseCredits = aiComponent * 10; // Scale to meaningful numbers
  const finalCredits = baseCredits * 
    impactMultiplier * 
    scaleMultipliers[params.projectScale] * 
    verificationMultipliers[params.verificationLevel] * 
    timeBonus * 
    innovationBonus * 
    educationBonus * 
    engagementBonus;
  
  return Math.floor(finalCredits);
}
```

## 7.6 Governance Integration

### 7.6.1 Real-Time Voting Power Calculation

```typescript
function calculateVotingPower(
  cicBalance: number,
  participationHistory: ParticipationMetrics,
  reputationScore: number,
  blockchainTransactions: TransactionHistory
): number {
  // Base voting power
  const baseVotes = 1; // Every participant gets 1 vote
  
  // CIC-based voting power (capped to prevent plutocracy)
  const cicVotes = Math.min(Math.floor(cicBalance / 1000), 100); // Max 100 votes from CIC
  
  // Blockchain-verified participation power
  const verifiedVotes = Math.floor(
    (blockchainTransactions.proposalsSubmitted * 2) +
    (blockchainTransactions.votingHistory.length * 0.1) +
    (blockchainTransactions.reviewsSubmitted * 1.5)
  );
  
  // Participation-based voting power (legacy calculation)
  const participationVotes = Math.floor(
    (participationHistory.proposalsSubmitted * 2) +
    (participationHistory.votesCase * 0.1) +
    (participationHistory.projectsCompleted * 5)
  );
  
  // Reputation-based voting power
  const reputationVotes = Math.floor(reputationScore / 10);
  
  // Total voting power (with maximum cap)
  const totalVotes = baseVotes + cicVotes + participationVotes + reputationVotes;
  
  return Math.min(totalVotes, 1000); // Maximum 1000 votes per person
}
```
